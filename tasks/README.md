## 🚀 快速开始

### 开始第一个任务
复制 `01-基础模型-用户与角色.md` 的内容，喂给 AI 并说：
> 请执行这个任务

### 任务完成后
1. 根据验证检查点手动测试
2. 确认通过后，更新上方表格状态为 ✅
3. 继续下一个任务
